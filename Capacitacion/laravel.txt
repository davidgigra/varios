Comandos Laravel

Cambiar nombre del proyecto
php artisan app:name {{nombre}}


Crear Controlador Resources
php artisan make:controller TrainerController --resource

Activar Serve
php artisan serve

Añadir campo a una tabla
php artisan make:migration name_migration --table=table

ejecutar las migraciones faltantes
php artisan migrate

Crear autenticación
php artisan make:auth
